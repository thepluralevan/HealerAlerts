Place your custom sound files in this folder.
Supported formats: .ogg (recommended), .mp3

In the addon config panel, enter just the filename (with extension) in the
"Sound file" field for any alert. Example: "ready.ogg"

You can also use subfolders: "Druid/swiftmend.ogg"

Note: WoW's sound engine requires files to be in the addon's directory at load time.
You must /reload after adding new sound files before they can be referenced.
